comandos de git importantes

    git init -> crea repositorio local
    git add -> extrae solo las modificaciones en este punto
    git add *.png -> agrega archivos de un solo tipo
    git add -A -> agrega todos los archivos que fueron modificados
    git status -> para chequear el estatus de todos los archivos
    git log -> presenta la informacion de los commits
    git reset nombrearchivo o *.extension -> para excluir archivos del stage
    git add "*.txt" -> agrega todos los archivos txt de todo el proyecto
    git add *.txt -> agrega todos los txt en el directorio actual
    git add --all -> agrega todos los archivos
    git add pdfs/*.pdf -> agrega todos los PDFS dentro de la carpeta PDFS
    git add pdfs/ -> agrega todos los archivos dentro de la carpeta pdf
    git commit -m
    git checkout -- . reconstruye el proyecto a como estaba en el ultimo commit
    git reset --hard 516be15 -> retroceder en el tiempo borrando archivos el numero es el id de commit
    git reflog -> lista las acciones realizadas en el repositorio
    git mv destruir-mundo.txt salvar-mundo.txt -> para cambiar nombres de archivos
    git rm salvar-mundo.txt
    git add -u -> para actualizar
    git branch ramanombre-> para crear ramas
    git branch -> para saber en que rama estamos
    git checkout nombrerama-> para movernos a una rama
    git commit -am "texto" -> para hacer commit y agregarlo al stage
    git diff rama-villanos master -> para checkear los cambios en lasramas
    git merge -> para unir ramas
    git branch -d nombrerama -> para eliminar una rama
    git checkout -b nombrerama -> para crear rama y movernos automaticamente
    git tag nombre-> para crear tags
    git tag -> listar tags
    git tag -d -> para eliminar un tag
    git tag -a v1.0.0 -m "el mensaje de todos los los mensajes" -> para hacer tags con msg
    git tag -a v0.1.0 5ba149b -m "la locura" -> para poner un tag en un commit specifico
