comandos de git importantes

    git init -> crea repositorio local
    git add -> extrae solo las modificaciones en este punto
    git add *.png -> agrega archivos de un solo tipo
    git add -A -> agrega todos los archivos que fueron modificados
    git status -> para chequear el estatus de todos los archivos
    git log -> presenta la informacion de los commits
    git reset nombrearchivo o *.extension -> para excluir archivos del stage
    git add "*.txt" -> agrega todos los archivos txt de todo el proyecto
    git add *.txt -> agrega todos los txt en el directorio actual
    git add --all -> agrega todos los archivos
    git add pdfs/*.pdf -> agrega todos los PDFS dentro de la carpeta PDFS
    git add pdfs/ -> agrega todos los archivos dentro de la carpeta pdf
    git commit -m
    git checkout -- . reconstruye el proyecto a como estaba en el ultimo commit
